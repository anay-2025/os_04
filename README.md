# Parallel Cramer's Rule Solver — OS Project

## Project Title
Multithreaded Solution of Linear Equations using Cramer's Rule

## Description
This project solves a system of linear equations using **Cramer's Rule** by computing
determinants through a **recursive algorithm**. Multiple child processes are created
using **fork()** to perform determinant calculations in parallel and analyze execution
performance on multi-core systems.
